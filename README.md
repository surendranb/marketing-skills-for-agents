# 🎯 Marketing Skills for Agents

A curated collection of high-fidelity marketing frameworks codified into AI Agent Skills. Instead of generic prompts, these skills use authoritative mental models from the "Giants" of marketing to transform agents into strategic partners.

## 🏛️ The Lattice of Mental Models

This library is organized by the frameworks of marketing legends:

| Framework | Author | Status | Description |
| :--- | :--- | :--- | :--- |
| **Obviously Awesome** | April Dunford | ✅ Active | Systematic 5-component product positioning. |
| **Influence Auditor** | Robert Cialdini | ✅ Active | Psychological audit using the 7 Principles of Persuasion. |
| **Big Idea Engine** | David Ogilvy | ✅ Active | Research-backed advertising and USP discovery. |
| **Purple Cow** | Seth Godin | 🛠️ Pending | Remarkability and spreadability audit. |
| **SB7 StoryBrand** | Donald Miller | 🛠️ Pending | Customer-as-the-hero messaging clarification. |
| **Mental Availability**| Byron Sharp | 🛠️ Pending | Scale and distinctiveness framework. |

## 🚀 How to Use

These skills are designed to be used with Claude (via [Skills](https://docs.anthropic.com/en/docs/build-with-claude/agent-skills)) or any MCP-enabled agent.

1.  **Clone this repo.**
2.  **Point your agent to the `skills/` directory.**
3.  **Execute a skill by name** (e.g., "Use the Obviously Awesome framework to position my new SaaS").

## 🧪 Best Practices for Agent Skills

- **Methodology First:** We prioritize *process* over *output*. The agent is instructed on *how to think*, not just *what to write*.
- **Constraint-Driven:** We enforce strict rules (e.g., "No marketing-speak") to ensure high-fidelity strategic thinking.
- **Fact-Based:** Skills like the "Big Idea Engine" force the agent to mine for product facts before generating creative.
